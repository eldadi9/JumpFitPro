# 🏃‍♂️ JumpFitPro v2.1 - ניהול ירידה במשקל

## 📋 תיאור הפרויקט

**JumpFitPro v2.1** - אפליקציית PWA מקיפה לניהול תכנית ירידה במשקל באמצעות קפיצה בחבל. המערכת כוללת דשבורד מתקדם עם גרפים, מעקב קלוריות בזמן אמת, תכניות אימון מסודרות, אימון חי עם טיימר, ומעקב התקדמות במשקל.

**חדש ב-v2.1:** בחירת מין, תמונות פרופיל, שיתוף תכניות ב-WhatsApp/SMS/Email

**טכנולוגיות:**
- Backend: Hono (Cloudflare Workers)
- Database: Cloudflare D1 (SQLite)
- Frontend: HTML + TailwindCSS + JavaScript
- Charts: Chart.js (גרפים אינטראקטיביים)
- Icons: Font Awesome
- PWA: Service Worker + Manifest (התקנה למסך הבית)

## 📐 חישובים מדעיים

### נוסחת קלוריות (MET)

```typescript
calories_per_minute = 0.0175 * MET * weight_kg
total_calories = calories_per_minute * work_minutes
```

### רמות עצימות

| עצימות | MET | קפיצות לדקה | קלוריות לדקה (124 ק"ג) |
|--------|-----|-------------|-------------------------|
| קל     | 8.8 | < 100       | ~19 קלוריות            |
| בינוני | 11.8| 100-120     | ~25.6 קלוריות          |
| גבוה   | 12.3| 120-160     | ~26.7 קלוריות          |

### דוגמה חישובית (אלדד)
- משקל: 124 ק"ג
- עצימות: בינונית (MET 11.8)
- זמן: 10 דקות

```
קלוריות לדקה = 0.0175 * 11.8 * 124 = 25.6
סה"כ קלוריות = 25.6 * 10 = 256 קלוריות
```

### חישוב BMI
```typescript
BMI = weight_kg / (height_m * height_m)
```

דוגמה (אלדד):
- משקל: 124 ק"ג
- גובה: 193 ס"מ = 1.93 מ'
- BMI = 124 / (1.93 * 1.93) = 33.3
- מצב: עודף משקל

## 🖥️ API Endpoints

### משתמשים
- `POST /api/users` - יצירת משתמש חדש (כולל gender) ⭐ v2.1
- `GET /api/users` - קבלת כל המשתמשים (בסיסי)
- ✅ `GET /api/users-with-stats` - קבלת משתמשים עם סטטיסטיקות ומיון (v2.0)
- `GET /api/users/:id` - קבלת משתמש לפי ID
- `PUT /api/users/:id` - עדכון משתמש (כולל gender) ⭐ v2.1
- ✅ `DELETE /api/users/:id` - מחיקת משתמש (Soft Delete) (v2.0)
- ✅ `DELETE /api/users/:id/permanent` - מחיקת משתמש מלאה (Hard Delete) (v2.0)
- ✅ `PATCH /api/users/:id/favorite` - סימון/ביטול משתמש מועדף (v2.0)
- ⭐ `PATCH /api/users/:id/profile-image` - עדכון תמונת פרופיל (v2.1)

### תכניות אימון
- `GET /api/plans` - קבלת כל התכניות
- `GET /api/plans/:id` - קבלת תכנית לפי ID
- `GET /api/plans/:id/week/:week` - קבלת אימונים לשבוע ספציפי
- ⭐ `POST /api/plans/:id/share` - שיתוף תכנית ב-WhatsApp/SMS (v2.1)
- ⭐ `POST /api/plans/:id/email` - שליחת תכנית במייל (v2.1)

### אימונים
- `POST /api/workouts` - שמירת אימון חדש
- ✅ `PUT /api/workouts/:id` - עדכון אימון קיים (עם חישוב קלוריות מחדש)
- ✅ `DELETE /api/workouts/:id` - מחיקת אימון
- ✅ `POST /api/workouts/:id/duplicate` - שכפול אימון להיום
- `GET /api/workouts/user/:userId` - קבלת כל אימוני המשתמש
- `GET /api/workouts/user/:userId/stats` - סטטיסטיקות אימונים
